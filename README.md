# FraudShield AI

## Overview
FraudShield AI is a SaaS platform that empowers fintech companies, e-commerce platforms, banks, and payment gateways to detect fraudulent transactions in real-time using advanced machine learning.

## Target Audience
- Fintech Companies
- E-commerce Platforms
- Banks
- Payment Gateways

## Key Features
- **Upload Transaction Data**: Securely ingest large volumes of transaction data.
- **Real-time Fraud Prediction**: Instant analysis and scoring of transactions.
- **Fraud Alerts**: Automated notifications for suspicious activities.
- **Analytics Dashboard**: Comprehensive view and insights into fraud metrics.
- **REST API for Businesses**: Easy integration with existing systems.

## Project Architecture
The project is built with a microservices-inspired architecture:
- `frontend-react`: User interface (React).
- `gateway-nestjs`: API Gateway and routing (NestJS).
- `ml-backend-fastapi`: Machine Learning prediction service (FastAPI).
